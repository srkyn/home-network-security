# Security Policy

This repository documents a personal network security project in sanitized form.

## Sensitive Information

Please do not open issues or pull requests that include public IP addresses, private host inventories, firewall exports, credentials, VPN keys, certificates, or other sensitive operational details.

## Reporting Concerns

If you notice accidentally exposed sensitive information in this repository, open a minimal issue that identifies the file and line without repeating the sensitive value. If the concern requires private handling, contact the repository owner through the contact options on the GitHub profile.

## Scope

This is documentation for a defensive personal network project. It does not provide authorization to test, scan, or interact with any live network.
